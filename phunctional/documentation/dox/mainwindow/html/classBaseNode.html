Detailed Description

This is a singleton object that lists all possible nodes. The node objects held in this class are not drawn. Instead, they are kept as a toolbar reference. This class has been replaced with a simple vector inside the Canvas class

This defines the generic drawable object, which includes connectors (arrows and the like) and objects (such as a stick person or a state oval).
