This is a helper method for to_xml that sets an attribute in an XML node.

Parameters:
docThis is the document we are creating. Whatever intiates the save will create this object and will need to make it available in all delegations.
nodeThis is the node of the document. For example, when saving the nodes vector, we create a "Node" element.
labelThis is the name of the attribute we are saving.
valThis is the value associated with label. 
                                              {
  QDomElement el = doc.createElement(label);
  node.appendChild(el);
  QDomText text = doc.createTextNode(val);
  el.appendChild(text);
}
